Orientação a objetos é um modelo de análise , projeto e programação de software baseado na composição e interação entre diversas unidades chamadas de objetos , os programas são arquitetados através de objetos que relacionam-se entre si , uma classe é uma forma de definir um tipo de dado em uma linguagem orientada a objeto , é formada por dados e comportamentos , para definir os dados são utilizados os atributos ,e para definir o comportamento são utilizados métodos , depois que uma classe é definida podem ser criados diferentes objetos que utilizam a classe
